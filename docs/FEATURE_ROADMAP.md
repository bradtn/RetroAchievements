# RetroTracker Feature Roadmap

## Completed Features

### Free Tier
- [x] Login with RA username + API key
- [x] View profile (points, rank, recent games)
- [x] Browse games by console
- [x] Search games across all platforms
- [x] View achievement lists per game
- [x] Achievement rarity badges with percentage
- [x] Missable achievement indicators
- [x] Global leaderboards
- [x] Achievement of the Week
- [x] Compare users
- [x] Favorites list
- [x] Share cards (profile, game, achievement, comparison)
- [x] Light/dark theme support

### Premium Tier
- [x] Rarity badges on achievements
- [x] Detailed statistics page
- [x] Hardcore/softcore breakdown
- [x] Mastery tracker
- [x] Per-console mastered games
